网吧实时变声详细步骤

https://m.clonevoice.net 化音为乐，奏出美好世界

视频教程

大家好，今天分享如何在网吧实时变声。

变声可以男变女，女变男，抠脚大汉变萝莉。

变声需要耳机，而网吧刚好是有耳机和麦克风。

现在我们开始一步一步来吧

首先我们在百度搜索“在线实时变声”,

找到克隆声音网站，或者输入www.clonevoice.net

在网站底部找到“网吧实时变声手把手教程”,

点开后，我们今天就按照这个教程操作一遍。

一、安装 voicemeeter 虚拟声卡

下载：点击下载 voicemeeter

安装结束弹出一个对话框，提示要重启电脑，因为在网吧是不能重启电脑的，重启后会还原。没有重启电脑，在接下来会有一个额外的操作。

如果在家，建议要重启电脑。

二、设置电脑的麦克风和扬声器

分三点：

第一点：设置4万8的采样率；

鼠标移到电脑桌面右下角的声音"🔊"图标上，右击，弹出的菜单点击“声音(s)“,点击"录制"tab 页，双击设备进入设备属性页，选择"高级"tab 页，选择 48000 采样率。如下图：

播放和录制下面的每台设备，需要执行上图的 4、5、6、7 操作。

第二点：设置默认播放设备、设置播放设备非独占

设置本机扬声器为默认播放设备，选择"播放"tab，右击本机扬声器，选择默认设备，选择默认通信设备，效果如下图

设置播放设备非独占，双击默认的播放设备，选择“高级”，独占模式里不要勾选，确定。如下图：

第三点：选择默认录制设备

选择"录制"tab 页，右击 Voicemetter Output，选择默认设备，选择默认通信设备，效果如下图

三、配置虚拟声卡

打开第二步安装的 voicemeeter 虚拟声卡。执行以下 1~6 步。

四、变声客户端设置

下载安装：点击下载 chrome

用 chrome 浏览器登录 https://m.clonevoice.net

找到变声页面，点击开启变声服务，约 1 分钟，系统出现变声入口，点击进入。如下图

这时浏览器进入下图，会弹出申请试用麦克风权限提示，我们要点击同意。

1 选择输入设备，在列表中从上往下选择第一个设备,就是我们在用的物理麦克风，我这里是 USB 的麦克风

2 选择输出设备，在列表中从上往下选择第一个设备,找到 voicemeeter 和 input 文案的设备

3 点击最上方的开始变声

这样变声客户端就好了

五、语音通话变声设置

QQ 语音

实现 QQ 语音通话变声，需要在 QQ 的设置里下拉找到“音视频通话”，将麦克风设置为包含 voicemeeter 和 output 文案的设备即可。

游戏和直播

若变声到游戏和直播,则将麦克风（有的叫音频输入设备）设置含 voicemeeter 和 output 文案的设备即可。

六、常见问题

点击“开始变声”后，为什么听不到声音？

首先确定设备的连接都是对的；然后确定打开了voicemeeter客户端，如果没有打开，那么重新开启变声服务试试；
